Integrated Danger Detection App

"Eye Contact"

About the Project

Eye Contact is an innovative application designed to enhance user safety by detecting potential dangers in the environment. By leveraging cutting-edge technologies such as computer vision, spatial computing, and AR frameworks, IDEA aims to alert users to dangers through visual and auditory signals, minimizing distractions and ensuring a safer user experience.

Core Features

  • Danger Detection: Identifies hazards such as vehicles, obstacles, and environmental risks.
  • Real-Time Alerts: Provides visual, auditory, and haptic feedback to notify users of potential dangers.
  • Customizability: Users can tailor the application to their specific environments and needs.
  • Scalability: Designed for integration with smart cities and future technologies like Neuralink.

Technologies Used

  • Computer Vision: OpenCV, TensorFlow Lite, PyTorch Mobile.
  • AR Frameworks: ARKit, ARCore, MRTK.
  • Object Detection Models: YOLOv5, SSD, custom-trained models.
  • Audio Feedback APIs: Google Text-to-Speech, Amazon Polly.

Applications

  • Pedestrian Safety: Alerts for oncoming vehicles and obstacles.
  • Industrial Safety: Hazard detection for workers.
  • Accessibility: Assistance for visually impaired users.
  • Emergency Response: Dangers highlighted in disaster zones.

Development Roadmap

  1. Prototyping: Select hardware and build core functionalities using pre-trained models.
  2. Integration: Implement real-time alerts and spatial awareness capabilities.
  3. Testing: Conduct simulations and real-world testing to refine the application.
  4. Deployment: Package for target platforms and enable cloud-based updates.
